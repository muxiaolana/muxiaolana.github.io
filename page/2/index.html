td的自动换行设置

style=”word-wrap:break-word;word-break:break-all;width:xxxpx;”

一定要设置宽度

word-wrap:设置或检索当前行超过指定容器的边界时是否断开折行

word-wrap的取值有：

normal:默认值，允许内容超出容器边界；

break-word：内容在边界内换行。如果需要，词内换行（word-break）也将发生

word-break:设置或检索对象内文本的字内换行行为，尤其在多种语言时。

word-break的取值：

normal:默认值，只允许在此间还行；

break-all:该行为与亚洲语言的normal相同。也允许非亚洲语言文本行的任意字内断开，该值适合包含一些非亚洲文本的亚洲文本

keep-all:与所有非亚洲语言的normal相同。对于中文，韩文，日文不允许字断开。适合包含少量亚洲文本的非亚州文本。

function ifBrowser(){
    var Sys = {};
    var ua = navigator.userAgent.toLowerCase();
    var s;
    (s = ua.match(/msie([\d.]+)/))?Sys.ie = s[1]:
